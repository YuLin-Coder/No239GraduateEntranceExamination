## 项目介绍
基于springboot+vue的考研资讯平台管理系统：前端 vue2、element-ui，后端 maven、springmvc、spring、mybatis；角色分为管理员、学生；集成考研资讯、报考指南、资料信息、客服等功能于一体的系统。
